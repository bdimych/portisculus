


count-bpm-by-hands.sh
	OK -запускать mplayer на паузе
	-действия:
		space - счёт
			-считать и показывать bpm динамически
		OK -q - выход без bpm
			пауза + подтверждение
		d - done - выход с bpm
			пауза + подверждение
		r - reset counter
			пауза + подтверждение
		OK -p - pause
		OK -home/left/right - seek 0/-10/+10 seconds
	
		
		
		
		
		
		
		
		

-идея: если в базе вместо bpm например 'count by hands' то запускать ручной счёт
	т.е. управлять скриптом (1-create-bpm-database.rb) не из командной строки а через базу
